Изменения

IrrSimpleVoxelEngine, build 0.0.1: (-- ------- ----)

1) Добавлен hardware mapping hint. Сильный прирост FPS. При использовании данного хинта идёт довольно сильный расход памяти, так как hardware buffers в самом Irrlicht удаляются раз в 20 000 кадров. Для себя я перекомпилировал Irrlicht, где сделал это удаление раз в 60 кадров.
2) Добавлена упрощённая функция получения вокселя по трём точкам.
3) Изменена функция получения доступа к движку (и освобождение доступа).
4) Добавлена возможность мультитекстурирования и склейки вокселя из частей.
5) Исправлен небольшой баг невозможности установки логирования/не логирования некоторых действий.
6) Добавлена возможность разбивать чанк на произвольное количество мешей по желанию.
7) Сделан механизм работы с прозрачными объектами.
8) Исправлен Occlusion Query.
9) Добавлена возможность установки ID нодам чанков.
10) Добавлена возможность полного управления блоками чанков на уровне возможностей движка.
