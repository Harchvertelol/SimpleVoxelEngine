Документация по IrrSimpleVoxelEngine.

Введение

Враппер IrrSimpleVoxelEngine написан для отрисовки воксельного мира с помощью движка Irrlicht, сам же воксельный мир хранится и обрабатывается с помощью движка SimpleVoxelEngine.
