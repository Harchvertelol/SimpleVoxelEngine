Подробное описание

Адрес вокселя.

Данный класс хранит последовательность векторов, которые являются адресом. Последний вектор является вектором в последнем чанке. Предпоследний - вектором последнего чанка в предпоследнем чанке. И так далее. Например, если у нас простейший мир, где только чанки и кубики, размер чанка 16 на 16 на 16, то адрес кубика с глобальными мировыми координатами (0, 0, 0) будет следующим: (0, 0, 0) -> (0, 0, 0), а кубика с глобальными координатами (16, 0, 0) следующим: (1, 0, 0) -> (0, 0, 0).
