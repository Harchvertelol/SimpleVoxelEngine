Разделение строки в массив int.

Аргументы
masУказатель на map, в который будет записан результат.
sizeНеиспользуемый параметр. Можно ставить его любому числу. Например, нулю.
pointС какой точки начинать запись в массив. Чтобы запись началась с начала, необходимо положить point = 0.
strОбрабатываемая строка.
splitterПо какому символу производить разделение (длина splitter должна быть равна 1).
Возвращает
Возвращает длину получившегося массива.
